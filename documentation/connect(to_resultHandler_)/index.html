@available(iOS 13.0, *)
    open func connect(to tag: NFCTag, resultHandler: @escaping ((Result<Void, NFCReaderError>)) -> Void)

Connects the reader session to a tag and activates that tag.

A tag stays connected until your app connects to a different tag or restarts polling. Connecting to a tag that is already connected has no effect.

Parameters

Name Type Description
tag NFCTag

A tag to which the reader session should attempt to connect.
result​Handler @escaping ((Result<Void, NFCReader​Error>)) -> Void

A handler that the reader session invokes after the operation completes. The handler receives a Result with the cases:
success

A Void.
failure

An NFCReaderError object indicating that a communication issue with the tag occurred.
