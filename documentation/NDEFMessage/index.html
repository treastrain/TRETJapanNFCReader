Structure NDEFMessage 

public struct NDEFMessage

Initializers

init(records:​) 

public init(records: [NDEFPayload])

The length, in bytes, of the NDEF message when stored on an NFC tag.

The maximum length of an NDEF message is 128 KB. Creates an NDEF message with the specified records.

Parameters

Name Type Description
records [NDEFPayload]

An array of payload objects for the message. To create an empty message, pass in an empty array.

Creates an NDEF message from raw data representing the message.

Parameters

Name Type Description
data Data

A data object containing the raw bytes of a complete NDEF message. The data must contain only one NDEF message, and the message must contain NDEF payloads that are valid according to the NFC Forum NDEF RTD specification.
