## Availability
- iOS 9.0+
- iPadOS 9.0+
- macOS 10.10+
- Mac Catalyst 13.0+
- tvOS 9.0+
- watchOS 2.0+
- Xcode 12.5+
  - Swift 5.4+
- Xcode 13.0+ (if you want to use it with Concurrency)
  - Swift 5.5+ (if you want to use it with Concurrency)

**:warning:　Features related to NFC tag reading are only available in iOS 13.0+.　:warning:**
